CalendarEventsWidget -- example

Code To Embed

Copy this example code, then set the API access token, then embed the widget in your HTML page.

<html>
  <head>
    <title>Calendar demo</title>
    <link href="http://widgets.venuedriver.com/css/calendar.css"  rel='stylesheet' type='text/css'/>
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.6.4/jquery.min.js" charset="utf-8"></script>
    <script src="http://widgets.venuedriver.com/js/lib/date.js" charset="utf-8"></script>
    <script src="http://widgets.venuedriver.com/js/CalendarEventsWidget.js" charset="utf-8"></script>
  </head>
  <body>
    <div id='cal-test'>
    </div>
    <script>
      $('#cal-test').VenueCalendar({venue_id:1374})
    </script>
  </body> 
</html>
