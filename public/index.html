AccountEventsWidget

The AccountEventsWidget displays a list of upcoming ticket-sale events for a given account, with links to the ticket-sales page for each event.

To display the widget on your web page, copy our example JavaScript code, then set the API access token, then embed the widget in your HTML page. Here is a live demo of the widget.
